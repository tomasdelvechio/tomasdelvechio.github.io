]]> http://tomasdel.com/279/feed 0 Optimizar (un poco) el Firefox 3 o 4 http://tomasdel.com/146 http://tomasdel.com/146#respond Tue, 31 May 2011 18:40:06 +0000 http://blog2.tomasdelvechio.com.ar/?p=146 Este tip es viejo, incluso el comando que les paso es un robo de “acá”:http://ubuntulife.wordpress.com/2009/01/12/tip-compactar-las-bases-de-datos-sqlite-de-firefox-3/ pero esta bueno, y cada tanto lo uso:

bc. for f in ~/.mozilla/firefox/*/*.sqlite; do sqlite3 $f ‘VACUUM;’; done

Funcionamiento: El script recorre todos los archivos de la carpeta .mozilla/firefox (que es donde se encuentra instalado nuestros datos de Firefox) y limpia la base de datos interna. No modifica ninguna información de marcadores, historial, etc…, es una limpieza “interna”, no afecta en nada el uso del Firefox.

Como sugerencia, cierren el firefox antes de ejecutar el comando.

Vía: “Ubuntu Life”:http://ubuntulife.wordpress.com/2009/01/12/tip-compactar-las-bases-de-datos-sqlite-de-firefox-3/
