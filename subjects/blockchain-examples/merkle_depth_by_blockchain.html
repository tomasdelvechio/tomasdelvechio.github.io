Bitcoin Ethereum BSC Solana
Depth = ⌈log₂(transactions per block)⌉. A proof path carries exactly depth sibling hashes — so Bitcoin SPV proofs are ~12 hashes, while Ethereum proofs are only ~8. Ethereum uses a Merkle Patricia Trie (not a plain binary tree) for its world state, making depth harder to compare directly — the figure above refers to its transaction Merkle root only. Solana does not use a Merkle tree for intra-block transaction ordering the same way; it uses a different commitment scheme, so the figure shown is an approximation for the transaction hash tree.
